CalculateFixedRewards(int, IRandom, StakeRegularFixedRewardSheet, ItemSheet, int)

Calculates fixed item rewards for a given stake level. Fixed rewards have a predetermined item count (unlike rate-based rewards in CalculateRewards(Currency, FungibleAssetValue, int, int, StakeRegularRewardSheet, ItemSheet, IRandom)), and the tradable flag on each StakeRegularFixedRewardSheet.RewardInfo controls whether material items are created as tradable or non-tradable.

public static Dictionary<ItemBase, int> CalculateFixedRewards(int stakeLevel, IRandom random, StakeRegularFixedRewardSheet stakeRegularFixedRewardSheet, ItemSheet itemSheet, int rewardSteps)

Parameters

stakeLevel int

The current stake level used to look up the reward row.

Random source used for non-material item creation.

Sheet containing fixed reward definitions.

Sheet used to look up item metadata by ID.

The number of reward steps accumulated since the last claim. Each reward's count is multiplied by this value.

CalculateRewards(Currency, FungibleAssetValue, int, int, StakeRegularRewardSheet, ItemSheet, IRandom)

Calculates rate-based item and currency rewards for a given staking level. Unlike CalculateFixedRewards(int, IRandom, StakeRegularFixedRewardSheet, ItemSheet, int), reward quantities are derived from the ratio of the staked NCG amount to each reward's DecimalRate. Rewards whose computed quantity is zero or negative are skipped.
