Class ClaimWorldBossReward

Namespace
Nekoyume.Action

Lib9c.dll

The ClaimWorldBossReward class is an action that allows a player to claim rewards for their contribution in defeating a world boss. This action ensures that only the owner of the avatar can claim the reward and calculates the reward based on the player's contribution.
